MercadoLibre Chile scrapers

MercadoLibre Chile local stock scraper for Apify

The `jchame` MercadoLibre Chile actor family is built for `mercadolibre.cl` evidence: CLP prices, item IDs, canonical URLs, ratings, visible sold text, local/import signals, and MercadoLibre's own `Envio local` filtered search pages.

Which actor to use

  • Atomic Actor 1: production default for many queries, missions, retries, and local-only saturation.
  • Actor 1: simple checks, diagnostics, and small VIP detail batches.

Local-only input

{
  "searchQueries": ["cama perro"],
  "searchFilterMode": "envio_local",
  "maxItems": 60,
  "maxItemsPerQuery": 60,
  "maxPagesPerQuery": 1,
  "proxyConfiguration": {
    "useApifyProxy": true,
    "apifyProxyGroups": ["RESIDENTIAL"],
    "apifyProxyCountry": "CL"
  }
}

Fields to read

  • `local_stock_status`: local, import_only, or unknown.
  • `local_stock_evidence_text`: visible phrase such as `Llega gratis hoy`, `Internacional`, or `Envio desde China`.
  • `search_filter_mode` and `search_filter_applied`: confirm local filter mode was used.
  • `price_clp`, `review_rating_average`, `sold_quantity_text`, `permalink_canonical`.

Important competition rule

Imported MercadoLibre listings should not count as Chile local-stock competition. Failed or unknown visibility should stay unknown, not be treated as no competition.

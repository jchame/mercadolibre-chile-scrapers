MercadoLibre Chile scrapers

Scraper de MercadoLibre Chile con filtro Envio local

Estos actores de Apify estan especializados en `mercadolibre.cl`. Sirven para extraer evidencia publica de productos vendidos en Chile: precio en CLP, posicion, rating, texto de vendidos cuando aparece, URL canonica y senales de envio local o importado.

Actor recomendado

Para trabajos de produccion usa Atomic Actor 1. Para pruebas simples o diagnostico usa Actor 1.

Entrada para solo productos locales

{
  "searchQueries": ["cama perro"],
  "searchFilterMode": "envio_local",
  "maxItems": 60,
  "maxItemsPerQuery": 60,
  "maxPagesPerQuery": 1,
  "proxyConfiguration": {
    "useApifyProxy": true,
    "apifyProxyGroups": ["RESIDENTIAL"],
    "apifyProxyCountry": "CL"
  }
}

Como interpretar los datos

  • `local_stock_status = local`: hay evidencia visible de stock local o pagina filtrada por Envio local.
  • `local_stock_status = import_only`: hay evidencia visible de internacional, China, USA u otro origen externo.
  • `local_stock_status = unknown`: no hay suficiente evidencia visible.

Regla clave: lo importado no cuenta como competencia local en Chile. Lo desconocido tampoco debe convertirse automaticamente en oportunidad limpia.
